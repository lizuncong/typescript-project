### 类的定义
```typescript
class Person {
    name = 'lzc';
    getName(){
        return this.name;
    }
}
```

### 类的继承
可以重写父类的方法，使用super调用父类的方法
```typescript
class Person {
    name = 'lzc';
    getName(){
        return this.name;
    }
}
class Teacher extends Person{
    getTeacherName(){
        return this.name
    }
    getName(){
        super.getName();
        return '重写'
    }
}
```

### 类中的访问类型和构造器
- 类中的访问类型
    + private：只能在类内访问，在继承的子类中不能使用
    + protected：允许在类内及继承的子类中使用
    + public。默认的属性类型就是public。允许在类的内外访问属性

```typescript
class Person{
    name: string | undefined;
    private age: number | undefined; // 只能在Person类内访问，不能通过实例访问
    protected address: string;
    printName(){
        console.log('my age is：', this.age)
        console.log('my name is：', this.name);
    }
}

class Teacher extends Person{
    printInfo(){
        console.log('不能访问age：', this.age);
        console.log('可以访问address：', this.address);
        console.log('可以访问name：', this.name);
    }
}

const p = new Person();
p.name = 'lzc';
console.log('p.name..', p.name);
p.printName();

```

### 构造函数
构造函数的普通写法
```typescript
class Person{
    name: string;
    private age: number; // 只能在Person类内访问，不能通过实例访问
    protected address: string;
    constructor(name: string, age: number, address: string){
        this.name = name;
        this.age = age;
        this.address = address
    }

    setAge(age: number){
        this.age = age;
    }
    
    printName(){
        console.log('my age is：', this.age)
        console.log('my name is：', this.name);
        console.log('my address is：', this.address);
    }
}

const p = new Person('lzc', 26, 'beijing')
p.printName();
```
构造函数的简洁写法，一般采用这种方式多点
```typescript
class Person{
    constructor(public name: string, private age: number, protected address: string){
        this.name = name;
        this.age = age;
        this.address = address
    }

    setAge(age: number){
        this.age = age;
    }

    printName(){
        console.log('my age is：', this.age)
        console.log('my name is：', this.name);
        console.log('my address is：', this.address);
    }
}

const p = new Person('lzc', 26, 'beijing')
p.printName();
console.log('p.name', p.name);

```

### 继承与构造函数
```typescript
class Person{
    constructor(public name:string){

    }
}

// 子类没有实现 constructor
class Teacher extends Person{
    printName(){
        console.log('my name is：', this.name);
    }
}

const tea = new Teacher('lzc');
tea.printName();

```
如果子类实现了构造器，则一定要在子类中手动调用父类的构造器
```typescript
class Person{
    constructor(public name:string){}
}

class Student extends Person{
    constructor(private age: number, name: string){
        // 一定要手动调用父类的构造器
        super(name);
    }
    printInfo(){
        console.log('my name is：', this.name);
        console.log('my age is：', this.age);
    }
}

const stu = new Student(26,'lzc');
stu.printInfo();

```
